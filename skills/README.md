# skills

CLAUDE.md-compatible skills — one per adapter. Each skill tells a coding agent how to install, configure, and integrate the CELLO adapter for a specific agent variant.

Skills are also submitted to the OpenClaw skill registry so they're discoverable directly from within supported agents.

| Skill | For |
|---|---|
| `openclaw.md` | OpenClaw |
| `nanoclaw.md` | NanoClaw |
| `paperclip.md` | Paperclip |
| `ironclaw.md` | IronClaw |
| `zeroclaw.md` | ZeroClaw |
| `hermes.md` | Hermes Agent |
| `nanobot.md` | NanoBot |
| `picoclaw.md` | PicoClaw |

## Status

Pre-implementation. Skills will be written alongside their corresponding adapters.
