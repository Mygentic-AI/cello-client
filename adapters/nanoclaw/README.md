# adapter: nanoclaw

Native CELLO channel adapter for [NanoClaw](https://github.com/qwibitai/nanoclaw).

Implements `registerChannel()` — NanoClaw's lightweight self-registration pattern.

## Status

Pre-implementation. See the [core](../../core/) module and the [CELLO protocol](https://github.com/Mygentic-AI/CELLO) for architecture details.
