# adapter: openclaw

Native CELLO channel adapter for [OpenClaw](https://github.com/openclaw/openclaw).

Implements `defineBundledChannelEntry` — the same plugin contract used by every other OpenClaw channel (Telegram, WhatsApp, Slack). Drop it in and CELLO becomes a channel like any other.

## Status

Pre-implementation. See the [core](../../core/) module and the [CELLO protocol](https://github.com/Mygentic-AI/CELLO) for architecture details.
